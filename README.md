## Local Development

No build step required. Clone and open directly in a browser:

```bash
# Clone the repo
git clone https://github.com/kavan-patel05/kavanpatel.dev.git

# Open in browser (Windows)
start index.html
```

## Key Features

- Custom animated cursor with ring follow (desktop)
- Animated hero with ambient orb background
- Scroll-reveal animations via IntersectionObserver
- Frosted glass navigation with scroll-lock sticky state
- Responsive mobile layout with slide-out menu
- AJAX contact form with live submit feedback
- Dynamic copyright year
